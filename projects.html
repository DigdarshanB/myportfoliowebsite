Desktop App using Java

This is a desktop application that I made using Java Swing. The program has text fields to input data and is able to store the entered data in an array list and suitable function is carried out when respective buttons are clicked. For example, if I press the Clear button, the texts from all the text fields are removed. Similarly, all the buttons have their own functions. Please check my git hub for the whole code.

Land Rental System

This is a land rental system that I made using Python. The data of availabe lands is read from a CSV file. I have made 4 modules for code reuse and for the neatness of code. With each transaction, an invoice/bill file is generated that includes customer's details, time of rent/return and so on. All the modules are imported on the main file and ran from the main file in Python's default IDLE.
